# 리포트 출력 템플릿

아래 형식으로 마크다운 리포트를 출력한다.

---

## 리뷰 대상 PR
- **PR**: (제목 + URL)
- **분석 코멘트**: 제출 X개 + ⏳ Pending Y개

---

## 한눈에 보기
**코멘트 N개** — 질문형 N · 제안형 N · 칭찬 N · 지적형 N (+ ⏳ Pending N개)

> (한 줄 리뷰어 스타일 진단)

---

## 전체 리뷰 패턴

**잘한 점**
- (패턴 설명 + 구체적인 코멘트 예시 인용)

**개선이 필요한 패턴**
- (패턴 설명 + 해당 코멘트 목록)

|---|---|

## 작성 원칙

- **개선 코멘트 상세**는 명확히 문제가 있는 것만 (전체의 50% 이하)
- **잘한 점 반드시 포함** — 없으면 분석 불균형
- **놓친 관점**은 실제 diff 기반만, 추측 금지
- **⏳ Pending** 코멘트는 submitted와 동일하게 분석에 포함하되 `⏳` 표시 유지
